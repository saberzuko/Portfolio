Computer Vision from my Perspective

Image Processing

Use Image Processing to pre-process and enhance the image. We should also use Image Processing to solve the most simple problems, which will lead to a reduction in computational cost.

Model Training

Models can be trained either using Machine Learning or Deep Learning. The decision of choosing the model should be based on the problem statement and the amount of data available.

Model Optimization

Major breakthrough requires model design changes. Hyperparameter tuning, usage of callbacks, and other methods can help to optimize the model. But in some cases, it is unavoidable to change the whole architecture.

Model Deployment

Deploy Optimized Models at Scale with Tensorflow and Flask APIs for both Edge and Cloud.
